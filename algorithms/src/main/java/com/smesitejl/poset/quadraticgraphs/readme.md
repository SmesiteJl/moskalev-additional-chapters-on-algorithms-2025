# QuadraticGraphs — разбиение на минимальное число монотонных графиков за O(n²)

## 1. Постановка задачи
Дан набор точек на плоскости. Требуется разложить эти точки на минимальное число непересекающихся монотонных графиков (возрастающих, неубывающих, убывающих или невозрастающих). В данном варианте требуется реализовать алгоритм с квадратичной сложностью O(n²).

---

## 2. Идея решения
Я свёл задачу разбиения точек на монотонные последовательности к задаче разбиения частично упорядоченного множества на минимальное число цепей. Для этого каждая точка рассматривается как вершина, а отношение порядка задаётся по координатам.

---

## 3. Выбор алгоритма
Для решения используется построение транзитивного ориентированного графа порядка и последующее нахождение минимального цепного разбиения. В квадратичном варианте проверка достижимости между вершинами производится полным перебором, что даёт сложность O(n²).

---

## 4. Алгоритм в общем виде
1. Считать все точки.
2. Для каждой пары точек определить, может ли одна следовать за другой в монотонном графике.
3. Построить ориентированный граф частичного порядка.
4. Последовательно находить максимальные цепи и удалять их из множества.
5. Количество найденных цепей и есть минимальное число графиков.

---

## 5. Корректность
Алгоритм на каждом шаге выделяет допустимую цепь в частично упорядоченном множестве. Поскольку каждая цепь соответствует монотонному графику, а удаление цепей продолжается до исчерпания всех вершин, в итоге получается корректное разбиение на монотонные графики. Минимальность следует из свойств цепного разбиения частичного порядка.

---

## 6. Асимптотическая сложность
Пусть n — количество точек.  
Проверка всех пар точек: O(n²).  
Построение и обработка графа: O(n²).  
Итоговая сложность: O(n²).  
Память: O(n²).

---

## 7. Особые случаи
- Одинаковые координаты точек.
- Полностью монотонный набор точек.
- Полностью несравнимые точки.
- Минимальное количество точек (n = 1).

---

## 8. Связь с теорией
Задача напрямую связана с разложением частично упорядоченных множеств на цепи и теоремой Дилворта. В квадратичном варианте используется прямой перебор без оптимизаций.

---

## 9. Что реализовано в программе
- Построение отношения порядка между точками.
- Квадратичное построение ориентированного графа.
- Последовательное выделение монотонных цепей.
- Подсчёт минимального числа графиков.
