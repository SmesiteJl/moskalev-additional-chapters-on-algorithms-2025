# GlassBalls — задача о стеклянных шариках

## 1. Постановка задачи
Дано k одинаковых стеклянных шариков и здание из N этажей. Шарик разбивается, если его уронить выше некоторого критического этажа. Требуется определить минимальное число бросков, которое гарантированно позволяет найти этот критический этаж в худшем случае. При этом N может быть очень большим.

---

## 2. Идея решения
Я рассмотрел, сколько этажей можно гарантированно проверить при фиксированном числе шариков и бросков. Это позволило перейти от поиска этажа к вычислению максимального количества проверяемых этажей за заданное число попыток.

---

## 3. Выбор алгоритма
Прямой перебор по этажам невозможен из-за больших значений N. Вместо этого используется биномиальная формула, задающая количество этажей, которые можно проверить за m бросков и k шариков. Поиск минимального числа бросков проводится с использованием бинарного поиска.

---

## 4. Алгоритм в общем виде
1. Зафиксировать число шариков k.
2. Для заданного числа бросков m вычислить количество этажей, которые можно гарантированно проверить.
3. С помощью бинарного поиска найти минимальное m, при котором это количество не меньше N.
4. Вывести найденное значение m.

---

## 5. Корректность
Для каждого набора параметров (k, m) биномиальная формула корректно учитывает все возможные исходы каждого броска — успешный и неуспешный. Бинарный поиск гарантирует нахождение минимального числа бросков, при котором выполняется условие покрытия всех этажей. Таким образом, алгоритм всегда возвращает правильный ответ.

---

## 6. Асимптотическая сложность
Пусть k — количество шариков, N — количество этажей.  
Вычисление покрытия для одного m: O(k).  
Бинарный поиск по m: O(log N).  
Итоговая сложность: O(k log N).  
Память: O(1) без учёта BigInteger.

---

## 7. Особые случаи
- k = 1 (линейный перебор).
- Малые значения N.
- Очень большие значения N, выходящие за пределы стандартных типов.
- Граничные случаи N = 0 и N = 1.

---

## 8. Связь с теорией
Используется классическая динамическая формула для задачи о стеклянных шариках, основанная на биномиальных коэффициентах и идее динамического программирования.

---

## 9. Что реализовано в программе
- Вычисление биномиального покрытия для заданного числа бросков.
- Бинарный поиск минимального числа бросков.
- Поддержка очень больших значений N за счёт BigInteger.
- Вывод минимального гарантированного числа попыток.
