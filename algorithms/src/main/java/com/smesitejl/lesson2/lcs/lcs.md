# НОП (LCS) — Наибольшая Общая Подпоследовательность

## 1. Постановка задачи
Даны две последовательности (строки).  
Нужно найти:

1. длину **наибольшей общей подпоследовательности** (LCS)
2. саму подпоследовательность

Подпоследовательность должна сохранять порядок символов в строках,  
но не обязана быть непрерывной.
---
## 2. Пояснение решения
Задача решается динамическим программированием:

- создается матрица `dp[n+1][m+1]`
- `dp[i][j]` — длина НОП для первых i символов строки 1 и первых j символов строки 2
- таблица заполняется слева направо, сверху вниз
- затем выполняется обратный проход (backtracking) для получения самой НОП
---
## 3. Формат ввода
Две строки:

1. первая последовательность
2. вторая последовательность

Пример:
```text
УНИВЕРСИТЕТ
УНИКС
```
---
## 4. Формат вывода
1. одно число - длина НОП
2. сама НОП
