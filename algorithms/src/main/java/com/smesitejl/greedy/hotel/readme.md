# Hotel — интервальное планирование заявок в гостинице

## 1. Постановка задачи
Задано n заявок на заселение в гостиницу. Для каждой заявки известны день заезда и день выезда. Гостиница получает фиксированную прибыль за каждую принятую заявку. Требуется выбрать максимальное число непересекающихся заявок, чтобы суммарная прибыль была максимальной.

---

## 2. Идея решения
Я свёл задачу к классической задаче выбора максимального числа непересекающихся интервалов. Для этого все заявки сортируются по возрастанию дня выезда и затем выбираются жадным образом.

---

## 3. Выбор алгоритма
Данная задача известна как задача интервального планирования, для которой сортировка по времени окончания и жадный выбор дают оптимальное решение. Более сложные подходы здесь не требуются.

---

## 4. Алгоритм в общем виде
1. Считать все заявки.
2. Отсортировать их по возрастанию времени выезда.
3. Последовательно выбирать заявку, если её начало не пересекается с последней выбранной.
4. Подсчитать количество выбранных заявок.
5. Умножить это количество на фиксированную стоимость.

---

## 5. Корректность
Выбор заявки с минимальным временем окончания оставляет максимальное количество времени для последующих заявок. Повторяя этот шаг, алгоритм строит множество непересекающихся интервалов максимального размера, что гарантирует максимальную прибыль.

---

## 6. Асимптотическая сложность
Пусть n — количество заявок.  
Сортировка: O(n log n).  
Основной проход: O(n).  
Итоговая сложность: O(n log n).  
Память: O(n).

---

## 7. Особые случаи
- Полностью пересекающиеся заявки.
- Несколько заявок с одинаковыми датами.
- Отсутствие допустимых заявок.
- Единственная заявка.

---

## 8. Связь с теорией
Используется классический жадный алгоритм интервального планирования, который изучается в курсе алгоритмов как пример корректного применения жадной стратегии.

---

## 9. Что реализовано в программе
- Сортировка заявок по дате выезда.
- Жадный выбор непересекающихся интервалов.
- Подсчёт максимального количества заявок.
- Вычисление общей прибыли гостиницы.
