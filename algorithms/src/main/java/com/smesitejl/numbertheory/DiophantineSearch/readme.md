# DiophantineSearch — поиск натуральных решений диофантовых уравнений

## 1. Постановка задачи
Дано линейное диофантово уравнение вида a·x + b·y = c.  
Требуется определить, существуют ли натуральные решения (x, y), и при наличии — найти одно из них.

---

## 2. Идея решения
Я использовал расширенный алгоритм Евклида для поиска одного частного решения уравнения a·x + b·y = gcd(a, b).  
Далее решение масштабировалось до c, и проверялось, существует ли набор натуральных x и y после корректировки общего решения.

---

## 3. Выбор алгоритма
Расширенный алгоритм Евклида является стандартным и быстрым методом нахождения решения линейных диофантовых уравнений. Он позволяет находить частное решение за время O(log(max(a, b))).

---

## 4. Алгоритм в общем виде
1. Вычислить d = gcd(a, b).
2. Проверить, делится ли c на d.
3. Найти частное решение (x₀, y₀) с помощью расширенного Евклида.
4. Умножить частное решение на (c / d).
5. Подобрать параметр t для получения натуральных x и y.
6. Если такие t существуют — вывести решение.

---

## 5. Корректность
Расширенный алгоритм Евклида корректно находит частные решения линейного уравнения.  
Все остальные решения выражаются линейным образом, что позволяет перебором по параметру t находить натуральные решения, если они существуют.

---

## 6. Асимптотическая сложность
Время: O(log(max(a, b)))  
Память: O(1)

---

## 7. Особые случаи
- Отсутствие натуральных решений.
- Нулевые коэффициенты.
- Уравнение вида a·x = c.
- Большие коэффициенты.

---

## 8. Связь с теорией
Используются классические методы теории чисел: gcd, расширенный алгоритм Евклида, описание общего решения линейного диофантова уравнения.

---

## 9. Что реализовано в программе
- Алгоритм Евклида и расширенный вариант.
- Проверка делимости c на gcd(a, b).
- Конструирование общего решения.
- Поиск натуральных решений.
