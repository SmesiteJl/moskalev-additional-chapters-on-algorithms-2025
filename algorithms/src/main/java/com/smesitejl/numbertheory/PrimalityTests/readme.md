# PrimalityTests — тест Ферма, тест Миллера–Рабина и числа Кармайкла

## 1. Постановка задачи
Требуется реализовать несколько тестов простоты:
- тест Ферма,
- вероятностный тест Миллера–Рабина,
- проверку числа на принадлежность к числам Кармайкла.

---

## 2. Идея решения
Я использовал модульное возведение в степень и свойства чисел, удовлетворяющих условию Ферма. Для Миллера–Рабина реализована проверка по представлению n−1 = 2^s · d. Для чисел Кармайкла используется их определение через обманчивость теста Ферма.

---

## 3. Выбор алгоритма
Тест Миллера–Рабина считается лучшим вероятностным тестом простоты для больших чисел. Тест Ферма добавлен как более простой базовый вариант. Проверка Кармайкла — как дополнительная часть теории.

---

## 4. Алгоритм в общем виде
1. Реализовать быстрое модульное возведение в степень.
2. Для теста Ферма проверить условие a^(n−1) ≡ 1 mod n.
3. Для теста Миллера–Рабина провести k случайных раундов.
4. Проверить критерий числа Кармайкла через делители n.

---

## 5. Корректность
Тест Миллера–Рабина является корректным вероятностным тестом, гарантирующим обнаружение составных чисел с высокой вероятностью.  
Тест Ферма корректен для всех чисел, кроме Кармайклов.  
Проверка Кармайкла использует их точное математическое определение.

---

## 6. Асимптотическая сложность
Для каждого теста:  
Время: O(k · log³ n)  
Память: O(1)

---

## 7. Особые случаи
- Малые числа.
- Числа Кармайкла.
- Чётные числа.
- Числа вида p^k.

---

## 8. Связь с теорией
Используются методы теории чисел: быстрые степени по модулю, малой теорема Ферма, критерии Миллера и Рабина, свойства Кармайкловых чисел.

---

## 9. Что реализовано в программе
- Быстрое возведение в степень (binary exponentiation).
- Тест Ферма.
- Тест Миллера–Рабина.
- Проверка числа Кармайкла.
- Тестирование различных входных данных.
