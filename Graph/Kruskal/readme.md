# Kruskal — алгоритм Краскала для построения минимального остовного дерева

## 1. Постановка задачи
Дан неориентированный взвешенный граф. Требуется построить его минимальное остовное дерево и найти его суммарный вес.

---

## 2. Идея решения
Я отсортировал все рёбра по возрастанию веса и последовательно добавлял их в остов, если они не образуют цикл. Для проверки циклов использовалась структура непересекающихся множеств.

---

## 3. Выбор алгоритма
Алгоритм Краскала является классическим решением задачи минимального остовного дерева и хорошо сочетается с DSU для эффективной проверки циклов.

---

## 4. Алгоритм в общем виде
1. Считать все рёбра графа.
2. Отсортировать их по возрастанию веса.
3. Инициализировать структуру DSU.
4. Перебирать рёбра, добавляя допустимые в остов.
5. После получения n−1 ребра завершить работу.

---

## 5. Корректность
Алгоритм всегда добавляет минимальное возможное ребро, не образующее цикл. По теореме о жадном выборе для MST это гарантирует построение минимального остовного дерева.

---

## 6. Асимптотическая сложность
Пусть V — число вершин, E — число рёбер.  
Сортировка рёбер: O(E log E).  
Операции DSU: O(E α(V)).  
Итоговая сложность: O(E log E).  
Память: O(V).

---

## 7. Особые случаи
- Несвязный граф (остов строится по компонентам).
- Наличие рёбер с одинаковыми весами.
- Граф из одной вершины.
- Пустой граф.

---

## 8. Связь с теорией
Используется классический жадный алгоритм Краскала и структура непересекающихся множеств для контроля циклов.

---

## 9. Что реализовано в программе
- Чтение списка рёбер.
- Сортировка рёбер по весу.
- Реализация DSU.
- Построение минимального остовного дерева.
- Подсчёт его суммарного веса.
