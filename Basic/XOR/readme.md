# XOR — обмен значений с использованием операции XOR

## 1. Постановка задачи
Даны две целочисленные переменные. Требуется обменять их значения местами без использования дополнительной временной переменной.

---

## 2. Идея решения
Я использовал свойство операции XOR, позволяющее восстанавливать исходные значения при повторном применении.

---

## 3. Выбор алгоритма
XOR-обмен является классическим приёмом, демонстрирующим свойства побитовых операций и работу с представлением чисел в памяти.

---

## 4. Алгоритм в общем виде
1. Выполнить a = a XOR b.
2. Выполнить b = a XOR b.
3. Выполнить a = a XOR b.

---

## 5. Корректность
За счёт свойств XOR каждое из исходных значений восстанавливается корректно, и в итоге значения переменных полностью меняются местами.

---

## 6. Асимптотическая сложность
Время: O(1).  
Память: O(1).

---

## 7. Особые случаи
- Одинаковые значения переменных.
- Нулевые значения.
- Отрицательные целые числа.

---

## 8. Связь с теорией
Используются свойства побитовых логических операций и двоичного представления чисел.

---

## 9. Что реализовано в программе
- Реализация XOR-обмена значений.
- Проверка корректности результата.
- Вывод исходных и изменённых значений.
