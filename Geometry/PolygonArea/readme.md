# PolygonArea — площадь невыпуклого многоугольника

## 1. Постановка задачи
Дан набор из n пар вещественных чисел — координат вершин невыпуклого многоугольника, заданных в порядке обхода. Требуется вычислить площадь данного многоугольника.

---

## 2. Идея решения
Я использовал формулу площади многоугольника через ориентированные площади треугольников, основанную на векторном и косом произведении. Суммирование производится по всем рёбрам многоугольника.

---

## 3. Выбор алгоритма
Использование формулы через косое произведение позволяет вычислить площадь за один линейный проход по вершинам. Более сложные геометрические алгоритмы здесь не требуются.

---

## 4. Алгоритм в общем виде
1. Считать координаты всех вершин многоугольника.
2. Последовательно для каждой пары соседних вершин вычислить слагаемое ориентированной площади.
3. Просуммировать все слагаемые.
4. Взять модуль от половины полученного значения.
5. Вывести результат.

---

## 5. Корректность
Формула через ориентированные площади корректно учитывает вклад каждого ребра в общую площадь многоугольника независимо от его выпуклости. Сумма ориентированных площадей даёт удвоенную площадь фигуры, а взятие модуля обеспечивает положительное значение результата.

---

## 6. Асимптотическая сложность
Пусть n — количество вершин многоугольника.  
Время: O(n).  
Память: O(1).

---

## 7. Особые случаи
- Минимальное число вершин (n = 3).
- Совпадение некоторых вершин.
- Нулевые координаты.
- Вырожденные случаи с нулевой площадью.

---

## 8. Связь с теорией
Используется классическая формула вычисления площади многоугольника через косое (векторное) произведение, изучаемая в вычислительной геометрии.

---

## 9. Что реализовано в программе
- Чтение координат вершин многоугольника.
- Вычисление ориентированных площадей для всех рёбер.
- Подсчёт итоговой площади.
- Вывод вещественного результата.
