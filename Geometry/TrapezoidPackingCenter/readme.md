# TrapezoidPackingCenter — безотходная укладка трапеций с вращением относительно центра

## 1. Постановка задачи
Дан набор трапеций, заданных координатами вершин. Разрешается вращать каждую трапецию относительно её центра (с перестановкой вершин A ↔ C и B ↔ D). Требуется определить, существует ли безотходная укладка всех трапеций в одну фигуру.

---

## 2. Идея решения
Я перебирал все возможные порядки трапеций и варианты их ориентации (повёрнута или нет). Для каждой полученной последовательности проверялось выполнение условий безотходности.

---

## 3. Выбор алгоритма
Поскольку количество трапеций невелико, допустим полный перебор всех перестановок с учётом двух ориентаций для каждой фигуры. Более быстрые алгоритмы требуют дополнительной геометрической структуры, которая в данной постановке отсутствует.

---

## 4. Алгоритм в общем виде
1. Считать все трапеции.
2. Сгенерировать все перестановки трапеций.
3. Для каждой перестановки перебрать все варианты поворота.
4. Проверить выполнение условий безотходной укладки.
5. При нахождении корректного варианта завершить работу.

---

## 5. Корректность
Алгоритм перебирает все возможные сочетания порядка следования трапеций и допустимых поворотов. Если существует хотя бы одно корректное решение, оно будет найдено. Если решение не найдено, то безотходная укладка невозможна.

---

## 6. Асимптотическая сложность
Пусть n — количество трапеций.  
Количество перестановок: O(n!).  
Для каждой — перебор ориентаций: O(2ⁿ).  
Итоговая сложность: O(n! · 2ⁿ).  
Память: O(n).

---

## 7. Особые случаи
- Совпадающие трапеции.
- Симметричные трапеции.
- Минимальное количество трапеций.
- Отсутствие допустимой укладки.

---

## 8. Связь с теорией
Задача относится к классу задач о переборе геометрических конфигураций с учётом симметрий и поворотов фигур.

---

## 9. Что реализовано в программе
- Хранение параметров трапеций.
- Генерация всех перестановок.
- Перебор ориентаций трапеций.
- Проверка условий безотходной укладки.
- Вывод одного допустимого варианта или сообщения об отсутствии решения.
